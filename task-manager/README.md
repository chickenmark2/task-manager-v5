# 課題管理アプリ v4

千葉工業大学の manaba から課題を自動取り込みして管理するアプリです。

## アプリ

> Vercel へのデプロイ後に URL を記載します

## v3 からの変更点

### 1. manaba 自動取得の改善
- **タブを問わずスキャン可能に** — 以前は拡張機能のポップアップを開いた時に「アクティブなタブ」が manaba でないとスキャンできませんでした。v4 では Chrome 内で開かれている **すべての manaba タブ** を自動検索するため、課題管理アプリや別のページを見ながらでもスキャンできます。

### 2. 集中モード：ポモドーロ / 52-17 タイマー
- 集中モード開始時にタイマーの種類を選択できます。
  - **ポモドーロ**: 25 分集中 → 5 分休憩
  - **52/17 ルール**: 52 分集中 → 17 分休憩
- 集中中はカウントダウンのリングタイマーを表示。
- タイマーが 0 になると自動で休憩フェーズへ移行（ステップを完了しても休憩へ）。
- 休憩タイマーが 0 になると次の集中フェーズへ自動的に戻ります。

### 3. ステップ 0 件での課題登録
- ステップが 1 件しかない場合も × ボタンで削除できるようになりました。
- ステップなしで課題を作成・登録できます（シンプルなタスクに便利）。
- 0 ステップの課題は集中モードで「タスクを完了」ボタンが表示されます。

### 4. ダークモード
- ヘッダー右上の ☀️ / 🌙 スイッチでライト・ダークを切り替えられます。
- 設定はブラウザの `localStorage` に保存されるため、次回アクセス時も引き継がれます。
- ダッシュボード・課題作成・課題実行のすべての画面で切り替え可能です。

## 機能一覧

- manaba の課題を Chrome 拡張機能で手動取り込み
- 締め切り順・作成日順の並び替え
- 課題をステップに分割して進捗管理
- 集中モード（ポモドーロ / 52-17 タイマー付き）
- ご褒美タイムの設定
- ダークモード対応
- Firebase によるクロスデバイス同期

## Chrome 拡張機能のインストール

1. [Releases](../../releases) から `manaba-extension-v4.zip` をダウンロードして解凍
2. Chrome で `chrome://extensions` を開く
3. 右上の「デベロッパーモード」をオン
4. 「パッケージ化されていない拡張機能を読み込む」→ 解凍した `manaba-extension` フォルダを選択

## 使い方

### 課題のスキャン・登録

1. manaba（`https://cit.manaba.jp`）を **どこかのタブで開く**（アクティブでなくてもOK）
2. 拡張機能アイコンをクリックしてログイン（初回のみ）
3. 「🔍 manabaをスキャン」をクリック
4. 取り込む課題にチェックを入れて「✅ 選択した課題を登録する」をクリック

> **ポイント**: 未提出課題一覧ページ (`cit.manaba.jp/ct/home_library_query`) を開いておくと全課題をまとめて取り込めます。

### 集中モードの使い方

1. 課題をタップして課題詳細ページへ
2. 「🎯 集中モードで始める」をクリック
3. タイマーモードを選択（ポモドーロ / 52-17）
4. タイマーに従って集中・休憩を繰り返し、ステップを完了していく
